## 🎯 撮影仕様

### 環境設定

- **OBS Studio:** 最新版（30.x以降）
- **OS:** Windows 11 または macOS Sonoma
- **言語:** 日本語UI
- **設定:** 推奨設定を適用済み（本アプリで生成したOBS設定）

### 画像仕様

| 項目 | 仕様 |
|------|------|
| **解像度** | 1920x1080（元画像）|
| **Web表示サイズ** | 800x450（自動リサイズ）|
| **フォーマット** | WebP（next/imageが自動変換）|
| **圧縮品質** | 80% |
| **ファイル名** | ガイドIDに対応（例: `youtube-stream-key.webp`）|

### 撮影ガイドライン

1. **画面全体をキャプチャしない**
   - 設定ダイアログやメニューのみをフォーカス
   - 余計なウィンドウは閉じる

2. **重要箇所にハイライトを追加**
   - Figma、Photoshop、または画像編集ツールで赤枠を追加
   - 矢印やラベルで説明を補足

3. **個人情報をマスク**
   - ストリームキーは「●●●●●●●●」でマスク
   - デバイス名やユーザー名も必要に応じてマスク

4. **UI言語は日本語**
   - OBSの言語設定を日本語に変更してから撮影

## 📋 撮影対象リスト（10枚）

### 1. 必須設定（3枚）

#### `required/youtube-stream-key.webp`
- **画面:** OBS > 設定 > 配信
- **フォーカス:** 「サービス: YouTube」と「ストリームキー」入力欄
- **ハイライト:** ストリームキー入力欄を赤枠で囲む
- **マスク:** キー入力欄の内容を●●●でマスク

#### `required/microphone-setup.webp`
- **画面:** OBS > 設定 > 音声
- **フォーカス:** 「マイク音声」デバイス選択ドロップダウン
- **ハイライト:** デバイス選択欄を赤枠で囲む
- **補足:** 音声ミキサーのメーターも含める

#### `required/game-capture.webp`
- **画面:** ゲームキャプチャのプロパティダイアログ
- **フォーカス:** 「モード」ドロップダウン（「特定のウィンドウをキャプチャ」を選択状態）
- **ハイライト:** モード選択欄とウィンドウ選択欄を赤枠で囲む

---

### 2. パフォーマンス設定（3枚）

#### `performance/disable-preview.webp`
- **画面:** OBSメイン画面のプレビュー画面
- **フォーカス:** 右クリックメニューの「プレビューを有効化」チェックボックス
- **ハイライト:** チェックボックスを赤枠で囲む

#### `performance/process-priority.webp`
- **画面:** OBS > 設定 > 詳細設定
- **フォーカス:** 「プロセス優先度」ドロップダウン（「高」を選択状態）
- **ハイライト:** プロセス優先度欄を赤枠で囲む

#### `performance/disable-recording.webp`
- **画面:** OBS > 設定 > 出力 > 録画タブ
- **フォーカス:** 「録画を有効にする」チェックボックス（チェックを外した状態）
- **ハイライト:** チェックボックスを赤枠で囲む

#### `performance/output-mode.webp`
- **画面:** OBS > 設定 > 出力
- **フォーカス:** 「出力モード」ドロップダウン（「シンプル」を選択状態）
- **ハイライト:** 出力モード欄を赤枠で囲む

---

### 3. オプション設定（3枚）

#### `optional/windows-game-mode.webp`
- **画面:** Windows設定 > ゲーム > ゲームモード
- **フォーカス:** 「ゲームモード」トグルスイッチ（オン状態）
- **ハイライト:** トグルスイッチを赤枠で囲む

#### `optional/browser-hw-accel.webp`
- **画面:** OBS > 設定 > 詳細設定
- **フォーカス:** 「ブラウザのハードウェアアクセラレーションを有効にする」チェックボックス（チェック済み）
- **ハイライト:** チェックボックスを赤枠で囲む

#### `optional/audio-monitoring.webp`
- **画面:** オーディオの詳細プロパティダイアログ
- **フォーカス:** マイクの「音声モニタリング」ドロップダウン（「モニターと出力」を選択）
- **ハイライト:** 音声モニタリング欄を赤枠で囲む

1. **ローカルビルド:**
   ```bash
   npm run build
   ```

2. **開発サーバー起動:**
   ```bash
   npm run dev
   ```

3. **動作確認:**
   - ダウンロード後ガイド画面にアクセス
   - 各ガイド項目の「📸 画像で見る」ボタンをクリック
   - next/imageで画像が正しく表示されることを確認
   - WebP形式に自動変換されていることを確認（DevTools > Network）
